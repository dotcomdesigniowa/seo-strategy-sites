# Dotcom Design — SEO Strategy Sites

Auto-deployed to **seo-strategy.dotcomdesign.com** via Vercel.

## Adding a New Client

1. Create a folder named after the client's domain (e.g., `clientdomain.com/`)
2. Drop the generated strategy site files into that folder
3. Add a link to the client in `index.html`
4. Commit and push to `main` — Vercel deploys automatically
